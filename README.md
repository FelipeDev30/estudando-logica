**Estudando Lógica de Programação**

- **Objetivo:**: Reunir exemplos práticos e exercícios para aprender lógica de programação usando HTML, JavaScript e pequenos scripts didáticos.

**Estrutura do Repositório**
- **`Base/`**: Coleção de exercícios e desafios organizados por tema.
	- **`Dasafios/`**: Pequenos projetos e exercícios com páginas HTML/JS.
	- **`funções/`**: Exemplos e variações de funções JavaScript (com parâmetros e com retorno).
	- **`Study-Json/`**: Estudos e notas sobre JSON.
- **`Revisando/`**: Revisões de conceitos básicos — algoritmos, estruturas de controle, operadores, etc.

**Como usar / executar os exemplos**
- **Abrir no navegador:** A maioria dos exemplos são arquivos HTML com scripts; basta abrir os arquivos `index.html` no navegador.
- **Executar scripts JS isolados:** Você pode executar pequenos scripts com o Node.js (se instalado):

```
node caminho/para/o/arquivo.js
```

- **Exemplos interativos:** Caso queira testar alterações rapidamente, abra o arquivo HTML correspondente e use o console do navegador (F12).

**Destaques de arquivos**
- **`Base/Dasafios/calcularSalario/calcSalarial.js`**: Exemplo de cálculo salarial.
- **`Base/funções/functions-returns.js`** e **`Base/funções/functions-to.js`**: Uso de funções com e sem retorno.
- **`Revisando/estrutura-de-controle/if-else.js`**: Exemplos de estruturas condicionais.

**Como contribuir**
